# SC4079-FYP

## Overview
This repository contains the core research artifacts and scripts for my Final Year Project (FYP). The research focuses on the efficacy of Large Language Models (LLMs) in analyzing compiled binary files, specifically targeting vulnerability detection and reverse engineering.

I utilized the Juliet Test Suite as the primary dataset to provide a standardized environment for testing and evaluation.

## Repository Structure
**LLM_pipeline/:** Python scripts used to interface with the LLM, including prompt engineering and response parsing.
**exports/:** Binary artefacts exported via Ghidra (e.g., decompiled code, control flow graphs, and intermediate representations).
**ground_truths/:** The reference dataset used to validate LLM outputs against known vulnerabilities in the Juliet Test Suite.
